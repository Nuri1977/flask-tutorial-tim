# Setup guide

### Requirements:

- Postgres
- Python version `3.9^`
- IDE

## Setup

### Virtual env

Create virtual environment

`python -m venv venv`

activate the virtual env

`source venv/bin/activate` for linux and mac\
`venv\Scripts\activate` for windows

Install packages

`pip install -r requirements.txt`
`npm install`

Load CSS

`npm run tailwind`

```

## Run


```
python main.py
```
